German Documents

A easy and simple solution for Odoo documents.


Document's Configuration

  • Select European A4 as format.


  • Check the document template.



Text Template Configuration (optional)

  • Set default text in desired sector .




  • Set default text for quotation which can be translated.


  • Now create a quotation and select a customer. According to customer's language the default text is loaded.
