Invoice Send Type

Determine the way of sending the invoices to your customers.


Configuration

  • Select the way of sending an invoice for this customer.


  • Select "Email": User can send the invoice via email. (Hides the print button)
  • Select "Print": User can send the invoice via for e.g. post mail. (Hides the send by email button)
  • Now create an invoice and validate.
